# Vehicle Insurance Management System

## Overview

The Vehicle Insurance Management System is a Software Design and Analysis (SDA) project developed in C++.

The system simulates the workflow of an insurance company by managing customers, insurance policies, vehicles, claims, inspections, repair orders, and staff operations.

The project follows Object-Oriented Programming principles and demonstrates software design concepts through UML modeling and modular system architecture.

---

## Features

### Customer Management

* Register customers
* Store customer information
* Manage insured vehicles

### Insurance Policy Management

* Create insurance policies
* View policy details
* Maintain policy records

### Claim Processing

* Submit insurance claims
* Track claim status
* Manage claim information

### Inspection Management

* Vehicle inspection handling
* Surveyor assignment
* Inspection record management

### Repair Management

* Create repair orders
* Workshop coordination
* Repair tracking

### Staff Management

* Manager operations
* Salesman operations
* Surveyor operations

### File Handling

* Persistent data storage
* Record retrieval using file management
